# /frontend-design

Implementación mobile-first con filosofía estética.

## Checklist

### Pre-implementación
- [ ] Filosofía seleccionada documentada
- [ ] Tokens verificados
- [ ] Componentes existentes identificados
- [ ] Breakpoints definidos

### Implementación
- [ ] Mobile-first (375px base)
- [ ] Touch targets mín 44px
- [ ] Contraste AAA
- [ ] Motion respetando prefers-reduced-motion

### Post-implementación
- [ ] Review contra brief
- [ ] Test en múltiples viewports
- [ ] Validación de tokens

## Mobile-First Breakpoints

```css
/* Base: 375px */
/* sm: 640px */
/* md: 768px */

```

## Filosofía por Feature

| Feature | Filosofía |

| Canvas | Swiss (grid) |
| Pricing | Brutalist (transparencia) |
| Dashboard | Scandinavian |
