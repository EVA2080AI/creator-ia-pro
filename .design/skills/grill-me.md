# /grill-me

Interrogatorio sobre decisiones de diseño hasta resolver cada una.

## Propósito

Eliminar ambigüedad. Forzar claridad en decisiones de UI/UX.

## Preguntas Tipo

- ¿Por qué este color y no otro?
- ¿Qué usuario específico se beneficia de esto?
- ¿Qué pasa en mobile?
- ¿Esto cumple el principio [X] de nuestra filosofía?
- ¿Es esto accesible?
- ¿Hay una forma más simple?

## Uso

Cuando haya dudas sobre una decisión de diseño:
```
/grill-me [decisión]
```

## Output

- Decisiones documentadas
- Alternativas consideradas
- Justificación final
