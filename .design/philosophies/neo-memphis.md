# Neo-Memphis

> "Playful chaos. Clashing color. Anti-corporate."

## Principios

- **Caos juguetón** — Elementos dispersos intencionalmente
- **Color chocante** — Combinaciones inesperadas
- **Anti-corporativo** — Rompe reglas
- **Divertido** — No tomarse en serio

## Visual

- Formas geométricas simples
- Paleta neón/vibrante
- Composición asimétrica
- Elementos "pegados" con tape, etc.

## Aplicación en Creator IA

Uso muy limitado:
- Páginas de error 404
- Estados de celebración (confetti)
- Marketing ocasional
