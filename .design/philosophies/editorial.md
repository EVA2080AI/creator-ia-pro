# Editorial / Magazine

> "Content-led. Display typography. Print-inspired."

## Principios

- **Contenido primero** — El texto es el héroe
- **Jerarquía tipográfica** — Títulos grandes, cuerpo legible
- **Layout de revista** — Columnas, pull-quotes, etc.
- **Calidad de print** — Detalles finos

## Visual

- Títulos grandes y bold
- Columnas de texto
- Imágenes prominentes
- Espacio generoso
- Detalles tipográficos (drop caps, etc.)

## Aplicación en Creator IA

### Blog / Documentation
- Layout de lectura optimizado
- Tipografía grande y legible
- Líneas de 60-75 caracteres
- Jerarquía clara

### Ejemplo
```css
.article {
  max-width: 65ch;
  margin: 0 auto;
  
  h2 {
    font-size: var(--text-3xl);
    margin-top: var(--space-12);
    margin-bottom: var(--space-4);
  }
  
  p {
    font-size: var(--text-lg);
    line-height: var(--leading-relaxed);
    margin-bottom: var(--space-6);
  }
}
```
