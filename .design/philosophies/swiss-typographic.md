# Swiss / International Typographic Style

> "Grid-locked. Strong type hierarchy. Objective."

## Principios

- **Grid System** — Columnas y filas matemáticas
- **Sans-serif** — Helvetica, Inter, Akzidenz-Grotesk
- **Asymmetry** — Balance dinámico, no simétrico
- **Objective Photography** — Sin ornamentación
- **Clear Hierarchy** — Tamaño y peso indican importancia

## Grid System

```
Desktop: 12 columnas, 24px gutter, 1200px max-width
Tablet: 8 columnas, 16px gutter
Mobile: 4 columnas, 16px gutter
```

## Typography Scale

```
Display: 72px/700/0.95
H1: 48px/700/1.0
H2: 36px/700/1.1
H3: 24px/600/1.2
Body: 15px/400/1.65
Caption: 12px/500/1.5
```

## Aplicación en Creator IA

- Landing usa grid de 12 columnas
- Jerarquía tipográfica clara
- Espaciado matemático (8pt system)
- Elementos alineados al grid
