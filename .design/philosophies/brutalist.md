# Brutalist

> "Raw structure visible. Anti-polish. Content-first."

## Principios

- **Honest** — Muestra la estructura, no la esconde
- **Raw** — Sin suavizar bordes ni ocultar materiales
- **Direct** — Comunicación directa sin adornos
- **Transparent** — Pricing claro, sin letra pequeña

## Visual

- Bordes visibles y gruesos
- Colores planos sin gradient
- Tipografía bold y grande
- Sin sombras suaves
- Alto contraste

## Aplicación en Creator IA

### Pricing (Brutalist Transparency)
- Precios grandes y claros
- Sin "trucos" de marketing
- Todo upfront: features, costos, límites
- Comparativa directa

### Ejemplo
```css
.pricing-brutalist {
  border: 2px solid var(--zinc-900);
  background: white;
  padding: var(--space-8);
  
  .price {
    font-size: 3rem;
    font-weight: 900;
    letter-spacing: -0.02em;
  }
  
  .features {
    border-top: 1px solid var(--zinc-200);
    padding-top: var(--space-4);
    list-style: none;
  }
}
```
