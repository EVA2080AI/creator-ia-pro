# Genesis AI Studio - Creator IA Pro

## Project info

Genesis is a sovereign autonomous software engineering platform. This repository contains the core platform source code, optimized for high-fidelity application synthesis.

## How can I edit this code?

This project is fully decoupled and maintains its own technical authority.

**1. Clone the repository**
```sh
git clone <YOUR_GIT_URL>
```

**2. Install dependencies**
```sh
npm i
```

**3. Run Development Server**
```sh
npm run dev
```

## Internal Architecture & Export

Genesis uses the **Genesis Universal Schema** (v21.0) to export and synchronize artifacts across environments.

### Technologies
- **Core**: React 18 + Vite + TypeScript
- **UI Architecture**: Genesis Aether Design System (Vanilla CSS + Tailwind v4 base)
- **State & Logic**: Framer Motion + Lucide React
- **Backend Infrastructure**: Supabase v3 Ecosystem
